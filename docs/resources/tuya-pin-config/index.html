    Tuya Pinout Config

    Device configuration (user_param_key) can be extracted to JSON, using bk7231tools from a full firmware dump.

    Also see:

    • UPK2ESPHome

    Sources:

    • tuya_demo_light_pwm for Lights/bulbs
    • Original post by @blakadder on Discord channel #resources
